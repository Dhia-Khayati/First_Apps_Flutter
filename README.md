Tachaboh Game
A simple and engaging Flutter-based matching game inspired by the Tunisian concept of "Tachaboh" (similarity). Tap on one of the two images to change the other and try to make them match.
Important Note
This project is strictly for personal experimentation and learning purposes only.
It is not intended for production, commercial use, or public distribution.
Features

Clean and intuitive user interface
Real-time feedback ("You Win" / "Try Again")
Built with Flutter and Material Design
Easy to extend (e.g., add random images, scoring, levels)

Technologies

Flutter (Dart)
StatefulWidget for state management

How to Run

Clone the repository:Bashgit clone https://github.com/yourusername/tachaboh-game.git
cd tachaboh-game
Install dependencies:Bashflutter pub get
Run the app:Bashflutter run
